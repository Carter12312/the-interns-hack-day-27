# Bloomington Corporate HQ parking images

Corporate Headquarters lot images use these exact filenames in the parent
`parking_lot_samples/` directory:

```text
corporateHeadquartersA.jpg
corporateHeadquartersB.jpg
corporateHeadquartersC.jpg
corporateHeadquartersD.jpg
corporateHeadquartersE.jpg
corporateHeadquartersF.jpg
```

Each image must use a structure definition that matches its camera angle. The
current POC analyzer uses `api/data/parking_lot_structure.json`; replace that
single shared structure with per-lot structure files when the Corporate HQ
images and annotations are available.
